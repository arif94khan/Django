Let's talk about changing text, html content,and attributes with the DOM

To Edit styles, we've already seen we can use the .style tag. Now if we want to edit actual html or text or attributes we can use different methods.If you want to change the text, html content, or attributes you canuse the following:

  • myvariable.tesxContent- This returns the text
  • myvariable.innerHTML-This returns the actual html
  • myvariable.getAttribute()-This returns the original attribute
  • myvariable.setAttribute()-This allowed you to set an attribute
